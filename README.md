СтудДок - Система управления студенческими документами
Система для управления документами в учебных группах. Позволяет студентам и преподавателям обмениваться материалами, комментировать документы и отслеживать изменения.

Функциональность
📄 Лента документов - просмотр всех доступных документов с фильтрацией по категориям, дисциплинам и тегам

📤 Загрузка документов - создание новых документов с прикреплением файлов, выбором категории и видимости

👤 Личный кабинет - управление своими документами и просмотр истории версий

📚 Дисциплины - структурированное отображение документов по учебным дисциплинам

📅 Расписание - календарное отображение документов категории "Расписание"

🔔 Уведомления - система оповещений о новых документах и комментариях

⚙️ Администрирование - управление пользователями, группами, справочниками (для ADMIN и CURATOR)

Роли пользователей
👑 ADMIN - полный доступ ко всем функциям системы, управление пользователями и справочниками

📚 CURATOR - управление своей учебной группой, добавление/удаление студентов

👨‍🏫 TEACHER - загрузка и редактирование документов

🎓 STUDENT - просмотр, загрузка документов, комментирование

Требования к системе
Python 3.7 или выше

SQLite3 (встроенный в Python)

Операционная система: Windows / macOS / Linux

Установка и запуск
Вариант 1: Быстрый запуск
Скачайте файл app.py в удобную папку

Откройте терминал/командную строку

Установите зависимости:

bash
pip install flask
Запустите приложение:

bash
python app.py
Откройте браузер и перейдите по адресу: http://127.0.0.1:5000/
